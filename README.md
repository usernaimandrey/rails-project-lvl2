### Ruby on Rails 7

Deploy project locally:

1. clone this repository
  
  ```shell
  git clone https://github.com/usernaimandrey/rails-project-lvl2.git

  ```

2. go to directory

  ```shell
  cd rails-project-lvl2

  ```
  
3. install dependencies
  
  ```shell
  make setup

  ```

4. run

  ```shell
    make start
  ```

open on <http://0.0.0.0:3000/>
